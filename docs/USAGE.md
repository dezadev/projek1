# Dokumentasi Penggunaan Singkat

1. Login menggunakan email + PIN kasir (6-8 digit).
2. Masuk ke Dashboard untuk melihat total penjualan hari ini.
3. Kelola produk dari menu Produk: tambah/edit/hapus + barcode.
4. Transaksi POS:
   - Scan barcode atau input manual.
   - Atur qty, diskon, metode pembayaran (Tunai/QRIS).
   - Simpan transaksi (offline).
5. Cetak struk ke Bluetooth printer atau generate PDF lalu share WhatsApp.
6. Saat internet tersedia, jalankan proses sinkronisasi otomatis.

## Catatan UX
- Gunakan tombol aksi utama ukuran minimal tinggi 56dp.
- Prioritaskan flow kasir maksimal 3 klik untuk menambah item ke keranjang.
